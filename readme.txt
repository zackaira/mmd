== Installation ==

= Requirements =

MMD does not require any other plugins.


= Installation =

Installing MMD can be done either by searching for "MMD" via `Plugins > Add New` screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the `Plugins > Add New > Upload` screen in your WordPress dashboard
3. Activate the plugin through the `Plugins` menu in WordPress
